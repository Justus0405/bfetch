# bfetch
Another System information script for Linux written in bash

> [!INFO]
> You need to have some kind of NerdFont installed
> for the symbols to display properly!

## Installation
```shell
git clone https://github.com/Justus0405/bfetch.git
cd bfetch
chmod +x bfetch
./bfetch
```
## Usage
```plaintext
    usage:  bfetch [...]
    arguments:
        -h | --help
        -t | --temps
        -s | --system

```
