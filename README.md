# System-monitor
A System monitoring script for Linux written in bash

## Dependencies
```plaintext
bash
```

## Installation
```shell
git clone https://github.com/Justus0405/system-monitor.git
cd system-monitor
chmod +x system-monitor.sh
./system-monitor.sh
```
## Usage
```plaintext
    usage:  system-monitor [...]
    arguments:
        -h | --help
        -t | --temps
```
